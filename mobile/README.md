# ThreatTrack Mobile App

React Native mobile application for ThreatTrack.

## Setup

### Prerequisites
- Node.js v20+ (v24 recommended)
- For iOS: Xcode, CocoaPods
- For Android: Android Studio, Android SDK, JDK

### Installation

```bash
npm install
```

### Running the App

**iOS:**
```bash
cd ios
pod install
cd ..
npx react-native run-ios
```

**Android:**
```bash
npx react-native run-android
```

### Development

Start the Metro bundler:
```bash
npm start
```

## Features

- User authentication
- Threat reporting
- Real-time notifications
- Offline support
