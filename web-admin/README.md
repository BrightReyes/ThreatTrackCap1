# ThreatTrack Web Admin

Web admin panel for ThreatTrack built with React and Vite.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Access at: http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## Features

- Admin authentication
- User management
- Threat monitoring dashboard
- Analytics and reports
- Real-time notifications
